## 🎵 Música (passo obrigatório)
O site procura por um arquivo chamado **`ordinary.mp3`** na mesma pasta do `index.html`.

1. Pegue o seu arquivo MP3 da música *Ordinary*.
2. Renomeie para exatamente: `ordinary.mp3` (tudo minúsculo).
3. Coloque na mesma pasta/repositório do `index.html`.

A música começa a tocar quando a Thaís digitar a senha e abrir o site
(o clique no botão libera o áudio no navegador). Se o navegador bloquear,
basta tocar no botão ♪ no canto inferior direito.

## 🚀 Como publicar no GitHub Pages
1. Crie um repositório novo no GitHub (ex.: `nossa-historia`).
   - Dica: deixe **privado é possível?** Não — o GitHub Pages gratuito exige repositório **público**,
     mas a senha na entrada mantém a surpresa.
2. Envie para o repositório os 2 arquivos:
   - `index.html`
   - `ordinary.mp3`
3. No repositório, vá em **Settings → Pages**.
4. Em **Source**, escolha **Deploy from a branch**, branch **main**, pasta **/(root)** e salve.
5. Aguarde 1–2 minutos. O link ficará assim:
   `https://SEU-USUARIO.github.io/nossa-historia/`
6. Abra o link, digite a senha e teste tudo (fotos, música, corações) antes de enviar para ela. 💌
